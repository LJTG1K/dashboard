## The Problem: 10% Form-to-Signup

**Current Flow:**
1. User clicks ad → Sees Meta Lead Form
2. User fills form (email + basic info)
3. Form submitted → Zapier automation
4. Zapier sends email: "Welcome! Here's how to sign up on Sugargoo..."
5. **Only 10% click through and actually sign up**

**Likely Issues:**
- Email not compelling enough (doesn't motivate action)
- Sugargoo sign-up process too complex/confusing
- Lead quality too low (wrong audience)
- Timing issue (email arrives, user forgets)
- Missing social proof (why should they sign up?)
- Friction in sign-up flow (too many steps)
